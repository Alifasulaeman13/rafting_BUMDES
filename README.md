# Aplikasi Rapting - Sistem Booking Rafting

## Deskripsi Aplikasi
Aplikasi Rapting adalah sistem booking rafting berbasis web yang dikembangkan untuk memudahkan pengelolaan layanan rafting. Aplikasi ini dibangun menggunakan framework Laravel dan database MySQL.

## Fitur yang Sudah Diimplementasikan

### 1. Sistem Login
- Login khusus untuk admin dengan autentikasi yang aman
- Manajemen sesi dan keamanan akses

### 2. Dashboard Admin
- Tampilan dashboard yang informatif untuk admin
- Ringkasan data booking dan aktivitas terkini
- Navigasi yang mudah ke berbagai fitur pengelolaan

### 3. Manajemen Perahu
- Penambahan data perahu baru secara dinamis
- Pengelolaan informasi perahu (kapasitas, kondisi, dll)
- Pembaruan dan penghapusan data perahu

### 4. Manajemen Paket Rafting
- Pembuatan paket rafting secara dinamis
- Pengaturan harga, durasi, dan detail paket
- Pengelolaan ketersediaan paket

### 5. Manajemen Ojek Rafting
- Penambahan data ojek rafting secara dinamis
- Pengelolaan informasi ojek rafting
- Pengaturan ketersediaan ojek

### 6. Sistem Booking
- Proses pemesanan paket rafting
- Pencatatan data pelanggan
- Pengelolaan status booking

## Fitur yang Belum Diimplementasikan
- Sistem pembayaran online
- Integrasi dengan gateway pembayaran
- Konfirmasi pembayaran otomatis

## Cara Penggunaan
1. Login sebagai admin menggunakan kredensial yang telah disediakan
2. Kelola data perahu, paket rafting, dan ojek rafting melalui menu yang tersedia
3. Pantau dan kelola booking yang masuk melalui dashboard admin

## Teknologi yang Digunakan
- Laravel (PHP Framework)
- MySQL (Database)
- Bootstrap (Frontend Framework)
- JavaScript/jQuery

## Pengembangan Selanjutnya
- Implementasi sistem pembayaran online
- Pengembangan aplikasi mobile
- Penambahan fitur laporan dan analitik
